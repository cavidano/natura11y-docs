--- layout: documentation title: "Backdrops" ---

Backdrops are useful for placing a translucent image behind custom content. Backdrops offer two fundamentally distinct variations: an image proportional block that scales and stacks responsively, and a fixed-height backdrop that, when needed, adjust to the height of its inner content.

Accessibility Touchpoint

Backdrops replace CSS background images with inline HTML images so screen readers can read the image's alt text.

Learn more about accessible images.

Image Proportional

The default .backdrop selector requires two immediate children. The first is an image with a class of .backdrop__image. This image sets the size (proportional width and height) of our backdrop. The second direct child is a block-level element with a class of .backdrop__cover, inside of which custom content can be placed.

Breakpoint Modifiers

When using image proportional backdrops for featured content (like the full-width example below) the content may become too tall for the image on small viewports.

Include a stack breakpoint modifier (e.g. .backdrop--stack--md) to a .backdrop selector and the backdrop's image is placed above the content vertically until the viewport reaches the desired breakpoint specified in the modifier.
