Quickly prototype and build responsive layouts with a simple (and familiar) 12 column flexbox grid system.

Flexbox vs. CSS Grid Layout

In the spirit of inclusivity we must consider Internet Explorer 11 users. For this reason, our grid system continues to use Flexbox instead of CSS Grid Layout.

Future versions of our grid will likely use CSS Grid Layout, as soon as Internet Explorer 11 is discontinued (in 2025).

Grid Selectors

Grids require a parent selector with a class of .grid. Each child selector (or grid cell) requires a .grid__cell class (see Figure 1).

Padding and marging classes should not be added to .grid or .grid__cell selectors.

Grid Cell Widths

All .grid__cell selectors occupy 100% of the available width (stacked one above another) until otherwise specified‚Å†.

There are several ways to apply widths to grid cells. For equal width cells, apply the .grid--cells-{x} modifier class to the .grid selector, where x equals the desired width.

For example, to make a three-column grid using the 12 column system, apply .grid--cells-4 to .grid (12 total columns divided by 3 columns equals 4).
