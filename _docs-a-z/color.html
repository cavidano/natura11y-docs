Theme Colors

The following palette makes up your default theme colors.

With Natura11y, white is treated as a color and it serves as your default theme color. These five colors generate our theme color utilities, background color utilities, and border color utilities.

Customize all colors with CSS variables or Sass.

Theme Color Utilities

Probably one of Natura11y's coolest (and most useful) features are the theme color utility classes. Each theme color utility applies a predefined background color, text color, border color, link color, even warn and confirm colors to any selector!
